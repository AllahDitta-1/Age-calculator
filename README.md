# Age Calculator

A lightweight age calculator that takes a birth date and returns the age in years, months, and days using vanilla JavaScript date logic.

## Features

- Lets users select a date of birth from a browser date input.
- Calculates age with year, month, and day breakdown.
- Prevents selecting future dates by limiting the maximum date.
- Keeps the experience simple and beginner friendly.

## Tech Stack

- HTML
- CSS
- JavaScript
- Browser Date API

## Getting Started

- Clone or download this repository.
- Open index.html in your browser, or run the folder with VS Code Live Server.


## Notes

- Open index.html directly for local use.
- The calculation runs fully in the browser with no backend dependency.
